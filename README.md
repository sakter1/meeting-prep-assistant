# 🚀 Meeting Prep Assistant

AI-powered **Meeting Prep Assistant** built using Google AI Studio and Gemini models.

This project analyses meeting notes and presentation slides to generate structured meeting insights for stakeholders.

---

## 📂 Repository Contents

### 📁 prompts/
This folder contains all prompt versions used during development.

- Each file represents a different stage of prompt evolution:
  - `prompt_v1_baseline.md` → Initial version with basic instructions  
  - `prompt_v2_grounded_json.md` → Introduces structured JSON output and grounding rules  
  - `prompt_v3_strict_schema.md` → Final version with strict constraints and enhanced schema  

These prompts demonstrate **iteration, refinement, and improvements in output quality**.
